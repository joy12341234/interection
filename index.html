입장 감지

05 — 강의자료를 변기에서 내려

04 — 구현 성과

기획에서 구현까지

🗂️

강의자료 선택 시스템

6개 강의자료 zone 좌표 인식. 클릭 시 해당 영상이 마지막 프레임에서 정지 유지

🔥

LiDAR 이펙트 오버레이

클릭/드래그 위치마다 이펙트 영상 생성. FootstepBlackTransparent 셰이더로 검정 배경 투명 처리

🎤

MIC 이벤트 연동

실시간 RMS 분석으로 threshold 초과 감지. 소리지르기 zone에서만 발동되도록 모드 분리

✕

X버튼 메인화면 복귀

강의자료 창 X버튼 위치를 CLOSE zone으로 등록. Canvas sortingOrder로 레이어 충돌 해결

🎬

영상 인코딩 최적화

H.264 baseline profile + bt709 재인코딩으로 Unity VideoPlayer 루프 버그 해결

⚙️

config.json 기반 설계

빌드 없이 zone 좌표·영상·인터랙션 타입 변경 가능. 현장 세팅 유연성 확보
